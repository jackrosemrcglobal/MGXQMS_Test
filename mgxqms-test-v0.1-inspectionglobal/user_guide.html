Introduction

Welcome to the MRC Global Inspection Report Tool User Guide. This comprehensive tool has been designed to streamline the process of creating, managing, and submitting detailed inspection reports for valves, actuators, and other equipment.

The Inspection Report Tool provides a structured approach to documenting inspection activities, with specialized sections for different types of equipment and inspection parameters. This guide will walk you through all features and functionalities to help you use the tool effectively.

Key Features

  • Comprehensive inspection form with specialized sections for valves, actuators, and ancillaries
  • Image documentation capabilities with annotation
  • Save and load functionality for managing multiple reports
  • Export options for sharing and archiving reports
  • Customizable form layout through drag-and-drop functionality
  • Automatic calculations for time tracking
  • Integrated print functionality with optimized formatting

Getting Started

This section will guide you through the initial steps of accessing and using the Inspection Report Tool.

Accessing the Tool

  1. From the MRC Global Portal homepage, locate the "Inspection Report" card
  2. Click the "Open Report" button to launch the tool
  3. The tool will open in a new page with an empty inspection form ready to be filled out

System Requirements

The MRC Global Inspection Report Tool works best with:

  • A modern web browser (Chrome, Firefox, Edge, or Safari)
  • JavaScript enabled
  • Minimum screen resolution of 1280 Ã— 800 pixels (larger displays provide better user experience)
Note: For the best experience, use the tool on a desktop or laptop computer. While the interface is responsive, the complexity of the form makes it more challenging to use on smaller screens.

Interface Navigation

The Inspection Report Tool interface is organized into logical sections to facilitate easy navigation and data entry.

Header Section

At the top of the page, you'll find:

  • MRC Global Logo - Displayed in the top left
  • Title - "MRC Global Inspection Report"
  • Home Link - Click the home icon to return to the portal homepage
  • Form Actions Button - A dropdown menu with options for saving, loading, printing, and exporting reports

Control Number

Below the header, you'll find the Control Number section. This automatically generated identifier combines information from several fields:

  • Report Number
  • Supplier
  • IR Revision
  • Current Year
  • Date (in DD.MM.YYYY format)

Form Sections

The main form is divided into multiple sections, each focusing on a specific aspect of the inspection. Major sections include:

  • Basic Information
  • Contact Information
  • Order Information
  • Time Breakdown
  • Purpose and Summary
  • Item Details
  • Inspection Results
  • Equipment-specific inspections (Valves, Actuators, etc.)
  • Visual Inspection Notes
  • Document References
  • Image Documentation

Section Navigation

Each section has a header with a title and can be moved within the form using the drag handle (six vertical dots on the left side of each section header).

Tip: If the form is very long, you can use your browser's "Find" function (usually Ctrl+F or Cmd+F) to quickly locate specific sections or fields.

Filling Out Reports

This section provides guidance on how to complete the various sections of the inspection report form efficiently.

Basic Information

Start by filling out the basic information section:

  • IR Issued Date - Defaults to today's date but can be changed as needed
  • Reference Order/Project - Enter the reference order or project identifier
  • Report Number - Enter a unique report number
  • IR Revision - Enter the revision number (defaults to 0 for new reports)
  • Manufacturer - Enter the name of the equipment manufacturer
  • Supplier/SubSupplier - Enter the name of the supplier or subsupplier
  • MRC Client - Enter the name of the MRC client
  • Supplier Job No. - Enter the supplier's job number

As you fill out the Report Number, Supplier, IR Revision, and IR Date fields, the Control Number at the top of the form will automatically update.

Contact Information and Inspection Type

Complete the contact information section with:

  • Contact Person
  • Inspection Location
  • Tel No.

In the same section, select the appropriate checkboxes for:

  • Type of Inspection (Interim, Final, or Other)
  • Result of Inspection (Accepted, Hold, Rejected, or Other)
  • Order Status (Whole, Part, or Balance)

Time Tracking

The Time Breakdown section allows you to record time spent on different activities across the week:

  • Inspection Time
  • Reporting Time
  • Travel Time
  • Travel Km

Enter values for each day (M, T, W, TH, F) and the system will automatically calculate the partial totals and weekly total.

Equipment Inspection Sections

The tool includes dedicated inspection sections for different types of equipment:

  • Valves Inspection - For documenting valve-specific inspection parameters
  • Actuators Inspection - For documenting actuator-specific inspection parameters
  • Ancillaries Inspection - For documenting inspection of ancillary equipment
  • Spools/Framework Inspection - For documenting inspection of spools and frameworks

In each section, mark the appropriate radio button (Pass, Fail, or N/A) for each inspection item.

Note: Not all sections may be relevant for every inspection. Focus on completing the sections that apply to your specific inspection scenario.

Additional Notes & Comments

Use the "Additional Notes & Comments" section to provide any supplementary information about the inspection that doesn't fit in the structured fields.

Document References

The Document References section allows you to select which standards and specifications were used during the inspection. Simply check the boxes next to the relevant documents.

Saving & Loading

The Inspection Report Tool offers multiple options for saving and loading reports, ensuring you never lose your work.

Saving Reports

To save a report, click the "Form Actions" button in the top right corner and select one of the following options:

Save Report

This option saves the report to your browser's local storage:

  1. Click "Save Report" from the Form Actions menu
  2. Enter a name for your report when prompted
  3. Click OK to save

Note: Local storage is specific to your browser on your current device. Reports saved this way won't be accessible from other devices or browsers.

Save as File

This option saves the report as a JSON file that can be stored, shared, or loaded later:

  1. Click "Save as File" from the Form Actions menu
  2. Enter a name for your report when prompted
  3. Choose where to save the file when your browser's download dialog appears

Note: Saving as a file is recommended for long-term storage or when you need to share the report with others.

Loading Reports

To load a previously saved report, click the "Form Actions" button and select one of the following options:

Load Report

This option loads a report from your browser's local storage:

  1. Click "Load Report" from the Form Actions menu
  2. Select the report you want to load from the list that appears
  3. The form will be populated with the saved data

Load from File

This option loads a report from a previously saved JSON file:

  1. Click "Load from File" from the Form Actions menu
  2. Browse for and select the report file when prompted
  3. The form will be populated with the data from the file
Warning: Loading a report will replace all current form data. Make sure to save your current work before loading a different report.

Image Documentation

The Image Documentation section allows you to include visual evidence in your inspection reports.

Adding Images

  1. Navigate to the "Image Documentation" section of the form
  2. Click the "Add New Image" button
  3. Select an image file from your device (supported formats: JPEG, PNG, GIF)
  4. The image will be uploaded and displayed in the form

Managing Images

Each uploaded image includes:

  • A filename header
  • The image itself
  • A comments field where you can add notes about the image
  • A remove button to delete the image if needed

To add comments to an image:

  1. Locate the comments field below the image
  2. Enter your notes about what the image shows, relevant observations, etc.
  3. Comments are automatically saved as you type

To remove an image:

  1. Click the "Remove" button in the top right corner of the image item
  2. The image and its associated comments will be removed from the report
Tip: For the best results, use clear, well-lit images that highlight the specific aspects of the equipment being inspected. Adding detailed comments to each image helps provide context for later review.

Printing & Exporting

The Inspection Report Tool offers options for printing and exporting reports for documentation and sharing purposes.

Printing Reports

  1. Click the "Form Actions" button in the top right corner
  2. Select "Print Report" from the dropdown menu
  3. Your browser's print dialog will appear
  4. Select your desired printer and print settings
  5. Click Print to generate a hard copy of the report

When printing:

  • Control elements like the drag handles and form actions dropdown are automatically hidden
  • The layout is optimized for paper size
  • Font sizes are adjusted for better readability in print
Tip: If you want to save a digital copy rather than printing to paper, select "Save as PDF" or similar option in your print dialog. This creates a PDF version of the report that can be easily stored or shared electronically.

Exporting to CSV

For data analysis or integration with other systems, you can export report data to CSV format:

  1. Click the "Form Actions" button
  2. Select "Export to CSV" from the dropdown menu
  3. A CSV file with all form field data will be downloaded

The CSV export includes:

  • Field names and values in a comma-separated format
  • All text and numerical data from the form

If your report contains file attachments, they will be automatically bundled into a ZIP file that downloads alongside the CSV file.

Form Customization

The Inspection Report Tool allows you to customize the layout of the form to better suit your specific inspection workflows.

Reordering Sections

Most sections of the form can be reordered using drag and drop:

  1. Locate the drag handle (six vertical dots) on the left side of the section header
  2. Click and hold the drag handle
  3. Drag the section to its new position in the form
  4. Release to drop the section in place

The new order of sections will be preserved when you save the report.

Note: Some sections like the Control Number and Signatures sections cannot be reordered as they must remain at fixed positions in the form.

Layout Persistence

The tool automatically remembers your preferred section order:

  • When you save a report, the current section order is saved with it
  • When you load a saved report, the sections will be arranged according to the saved order
  • Your last used layout is also saved to your browser's local storage and applied to new reports
Tip: Consider organizing the form with your most frequently used sections at the top for easier access during inspections.

Tips & Tricks

Here are some helpful tips to make your experience with the Inspection Report Tool more efficient:

Quick Navigation

  • Use Ctrl+F (or Cmd+F on Mac) to quickly find specific fields or sections
  • Tab between fields to move through the form quickly
  • Organize sections with your most frequently used ones at the top

Data Entry

  • Fill out the Basic Information section first to generate the Control Number
  • For Time Breakdown, enter values as you go through the week for accurate tracking
  • When selecting radio buttons in inspection tables, click on the cell not just the radio button for easier selection
  • Use descriptive names when saving reports to make them easier to identify later

Working with Images

  • Resize large images before uploading for better performance
  • Write clear, concise comments for each image
  • Consider organizing images in a logical sequence (e.g., overview first, then details)

Saving Work

  • Save your work frequently to prevent data loss
  • Use "Save as File" for important reports or when work is complete
  • Use descriptive filenames that include date, project, and revision information

Optimizing for Print

  • Preview the print layout before printing to ensure all content fits properly
  • Consider using landscape orientation for better fitting of wide tables
  • If printing doesn't look right, try saving as PDF first, then print from the PDF viewer

Troubleshooting

Here are solutions to common issues you might encounter while using the Inspection Report Tool:

Issue Possible Cause Solution
Form does not load properly JavaScript disabled or browser compatibility issue
  • Ensure JavaScript is enabled in your browser
  • Try using a different modern browser (Chrome, Firefox, Edge)
  • Clear your browser cache and reload the page
Cannot save reports locally Local storage issues or private browsing mode
  • Make sure you're not in private/incognito browsing mode
  • Check if your browser has restrictions on local storage
  • Try using "Save as File" instead
Report does not print correctly Complex layout causing print formatting issues
  • Try printing to PDF first, then print the PDF
  • Adjust your printer settings (landscape mode may work better)
  • Try printing from a different browser
Cannot load a saved file File format issues or corrupted file
  • Ensure you're loading a file that was saved using the "Save as File" option
  • Check if the file extension is .json
  • Try loading a different saved report file
Image upload fails File size or format issues
  • Ensure the image is in a supported format (JPEG, PNG, GIF)
  • Reduce the image size if it's very large
  • Try uploading a different image to see if the issue persists
Control Number not generating Missing required fields
  • Ensure you've filled out Report Number, Supplier, and IR Revision fields
  • Verify the IR Date field has a valid date
Calculations not updating Input format issues
  • Ensure time values are entered as numbers
  • Avoid using text or special characters in numeric fields
  • Try clicking outside the field after entering a value
If Problems Persist: If you continue to experience issues that aren't resolved by these troubleshooting steps, try using a different browser or computer. If problems still persist, contact your system administrator for assistance.

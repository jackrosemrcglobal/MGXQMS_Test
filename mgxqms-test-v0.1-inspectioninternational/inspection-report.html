Release Notes - Quality Management Components

Version: 2.1.0

Release Date: December 2024

1. Overview

This release focuses on enhancing quality management features, including significant updates to the inspection reporting system, improved data accuracy, enhanced user interface, and expanded customization capabilities. The update streamlines workflows and provides better traceability for quality management processes.

2. New Features

  • Advanced Form Templates: Users can now create, save, export, and import custom form templates with specific section configurations for different inspection types.
  • Enhanced Image Documentation: Multiple image upload capability with detailed commenting system for comprehensive visual documentation.
  • Transposed Item Details: New transposed table layout for item details that allows better data organization and visibility.
  • Custom Inspection Sections: Fully customizable inspection categories and subsections with CSV import/export capabilities.
  • Google Translate Integration: Real-time language translation support for international inspection teams.
  • Dynamic Form Sections: Drag-and-drop reordering of form sections for personalized workflows.

3. Improvements

  • Enhanced Data Persistence: Improved local storage system with better error handling and data recovery capabilities.
  • Optimized Print Layout: Better formatting for printed reports with automatic scaling and professional appearance.
  • Streamlined Navigation: Consolidated header with better organization of tools and actions for improved user experience.
  • Responsive Design Updates: Enhanced mobile and tablet compatibility for field inspections.
  • Performance Optimization: Faster loading times and improved form responsiveness with large datasets.

4. Bug Fixes

  • Control Number Generation: Fixed automatic control number formatting issues with special characters in supplier names.
  • Time Calculation Accuracy: Resolved calculation errors in time breakdown totals when using decimal values.
  • File Save/Load Stability: Improved reliability of file-based save and load operations, especially with large inspection datasets.
  • Image Upload Handling: Fixed issues with large image files and improved error messaging for unsupported formats.
  • Form Section Visibility: Corrected issues where hidden sections would sometimes appear in printed reports.

5. Known Issues

  • Internet Explorer Compatibility: Limited functionality in Internet Explorer browsers. Workaround: Use Chrome, Firefox, Edge, or Safari for full functionality. Resolution expected in next major release.
  • Large Image Collections: Performance may be impacted when uploading more than 20 large images (>5MB each). Workaround: Compress images before upload or upload in smaller batches.

6. Deprecations & Removals

  • Legacy CSV Export: The old CSV export format has been replaced with an enhanced version that includes better data structuring and Unicode support.
  • Simplified Inspection Types: Removed redundant surveillance points and report type checkboxes to streamline the inspection workflow.

7. Security Updates

  • Enhanced Data Encryption: Implemented additional client-side encryption for sensitive inspection data stored in local browser storage.
  • Input Validation: Strengthened input validation to prevent XSS attacks and ensure data integrity.
  • File Upload Security: Enhanced file type validation and size restrictions for image uploads to prevent malicious content.

8. Compatibility

Browser Support:

  • Chrome 90+ (Recommended)
  • Firefox 88+
  • Safari 14+
  • Microsoft Edge 90+

Operating System:

  • Windows 10+
  • macOS 10.15+
  • Linux (Ubuntu 18.04+, CentOS 7+)
  • iOS 14+ (Safari Mobile)
  • Android 8+ (Chrome Mobile)

9. Upgrade Instructions

This web application automatically updates when you refresh your browser. To ensure you have the latest version:

  1. Save any current work using the "Save as File" feature
  2. Clear your browser cache (Ctrl+Shift+Del / Cmd+Shift+Del)
  3. Refresh the page (F5 / Cmd+R)
  4. Verify the new version by checking these release notes

Note: Existing saved reports and templates will be preserved during the update.
